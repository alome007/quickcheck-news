## Description

QuickCheck News App usecase

## Installation

```bash
$ yarn install

# auto-link packages
$ react-native link

#install pod
$ cd ios && pod install && cd ..

```

## Running the app

```
# Android
$ yarn android

# IOS
$ yarn ios
```
